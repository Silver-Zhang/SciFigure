# SciFigure Skill

Use SciFigure when a user wants an editable scientific PowerPoint figure from a sketch, screenshot, or reference style.

## Inputs

- reference sketch or current PPT screenshot;
- optional style reference image;
- user requirements;
- formula requirements;
- output filename;
- mode: review_mode or auto_mode.

## Outputs

- editable PPTX;
- generation script;
- preview image;
- asset decision table;
- check report;
- equation source file when needed.

## Core constraints

- preserve editability for text, shapes, arrows, panels, axes, and formulas where possible;
- use local generated images only after asset decision approval;
- keep scientific logic before visual decoration;
- keep style consistent with approved examples.
