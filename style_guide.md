# Style Guide

SciFigure uses clean academic slide styling.

## Default visual rules

- plain page background;
- clear title hierarchy;
- rounded panels;
- editable arrows and connectors;
- restrained academic color palette;
- local scientific illustrations only where needed.

## Color roles

- deep blue: titles, main arrows, and key outlines;
- light blue: data flow, water phase, or field influence;
- orange: core method or highlighted module;
- red: threshold or decision;
- gray: secondary border and placeholder.

## Fonts

Use a readable Chinese font when Chinese text is present. Prefer KaiTi or Microsoft YaHei on Windows.

## Avoid

- whole-slide raster images;
- very small labels;
- decorative icons without scientific meaning;
- cartoon or poster-like styles;
- unapproved image assets.
